# Math Toolkit

A collection of mathematics tools built while learning software development and mathematical computing.

This project serves as the foundation for a future AI-powered mathematics learning platform called **MathMind AI**.

---

## Current Features

### Equation Solvers
- Linear Equation Solver
- Quadratic Equation Solver
- Real Root Support
- Complex Root Support

### Software Features
- Interactive Command-Line Interface (CLI)
- Input Validation
- Error Handling
- Modular Code Structure
